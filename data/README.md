# Data

This directory contains generated data files produced by the simulation
notebooks. Files are not committed to version control as they can be
regenerated by running `notebooks/01_cohort_simulation.ipynb`.

## Files

| File | Description |
|------|-------------|
| `cohort.csv` | Base synthetic cohort (10,000 patients, demographics + DR status) |
| `predictions_tool_{a,b,c}_{equal,differential}.csv` | AI predictions for each tool × scenario combination |

## Generation

All data is synthetically generated using parameters drawn from published
NZ clinical and population studies. See `config/parameters.yaml` for the
full parameter specification with inline citations.

No real patient data is used at any stage.
